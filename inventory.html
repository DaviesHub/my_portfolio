Warehouse Inventory

Description

This program is built to assist shoes store managers keep track of their stock. The program stores details about each shoe in a txt file and reads from the file to retrieve information. The program contains functions that help the store manager search for a product, restock low quantity products, get the total price of each product and find the product with the highest quantity. This project makes use of object-oriented programming.

Package Requirements
